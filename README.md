## campagne-diane-ndeuna
Site vitrine de campagne Diane Ndeuna pour la COING

## Getting Started

### Prerequisites

- Node.js
- NPM
- TailwindCss

### Installing

```sh
npm install
```

### Running

```sh
npm run watch-css or npx tailwindcss -i ./src/assets/styles/input.css -o ./src/assets/styles/output.css --watch
```
